# JNDLink — สรุปการทำงาน

- ผู้ใช้สมัคร/ล็อกอิน
- ผู้ใช้สร้าง short URL ระบบสร้างรหัสสั้น `{code}` และคืนลิงก์ `https://{host}/s/{code}`
- เมื่อมีการเรียก `/s/{code}` ระบบค้น `original_url` เพิ่ม `visits` แล้วรีไดเร็กต์ไปยัง URL ต้นทาง
- `admin` ดู/แก้ไข/ลบลิงก์ทั้งหมดได้
- ข้อมูลเก็บใน MySQL, ส่วนหน้าใช้ Blade + Bootstrap
